# Landing_page_flutter

Landing page do Super Yama desenvolvida em Flutter.

## Funcionalidades

- Hero section com parallax
- Seção do Jornal da Semana
- Galeria de fotos
- Formulário de sugestões
- Localização
- Contato

## Como executar

```bash
flutter pub get
flutter run -d chrome
```

## Tecnologias

- Flutter
- Dart
- Material Design
